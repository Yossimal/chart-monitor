Connect Your Git Repository

Chart-Monitor requires a Git repository to load monitoring scripts. Follow the guide below to configure your SSH key and set the required environment variables.

GitHub – Deploy Key Setup

  1. Generate an SSH key pair:
    ssh-keygen -t ed25519 -C "chart-monitor-deploy" -f ~/.ssh/id_rsa_chart_monitor -N ""
  2. Copy the public key: cat ~/.ssh/id_rsa_chart_monitor.pub
  3. Go to your GitHub repo → Settings → Deploy keys → Add deploy key
  4. Paste the public key. Check "Allow write access" only if needed. Click Add key.
  5. Set environment variables: 
    GIT_SSH_URL=git@github.com:your-org/your-repo.git
GIT_SSH_KEY_PATH=/path/to/id_rsa_chart_monitor
GIT_TARGET_PATH=/app/store
SYNC_SECRET=your-strong-secret

GitLab – Deploy Key Setup

  1. Generate an SSH key pair:
    ssh-keygen -t ed25519 -C "chart-monitor-deploy" -f ~/.ssh/id_rsa_chart_monitor -N ""
  2. Copy the public key: cat ~/.ssh/id_rsa_chart_monitor.pub
  3. Go to your GitLab project → Settings → Repository → Deploy keys
  4. Click Add new deploy key, paste the public key, and save.
  5. Set environment variables: 
    GIT_SSH_URL=git@gitlab.com:your-group/your-repo.git
GIT_SSH_KEY_PATH=/path/to/id_rsa_chart_monitor
GIT_TARGET_PATH=/app/store
SYNC_SECRET=your-strong-secret

Bitbucket – Access Key Setup

  1. Generate an SSH key pair:
    ssh-keygen -t ed25519 -C "chart-monitor-deploy" -f ~/.ssh/id_rsa_chart_monitor -N ""
  2. Copy the public key: cat ~/.ssh/id_rsa_chart_monitor.pub
  3. Go to your Bitbucket repo → Repository settings → Access keys
  4. Click Add key, paste the public key, and save.
  5. Set environment variables: 
    GIT_SSH_URL=git@bitbucket.org:your-workspace/your-repo.git
GIT_SSH_KEY_PATH=/path/to/id_rsa_chart_monitor
GIT_TARGET_PATH=/app/store
SYNC_SECRET=your-strong-secret
Note: Restart the backend after setting the environment variables. This page will disappear once all four variables are properly configured.
